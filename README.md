📅 Sistema de Reservas

Projeto simples de sistema de reservas desenvolvido para gerenciar cadastros e operações básicas.

💡 Sobre

A ideia do projeto é simular um sistema onde é possível criar e gerenciar reservas, organizando dados de forma prática.
Serve como prática de lógica, organização de código e integração com backend.

⚙️ Funcionalidades

-Cadastro de reserva;
-Listagem de dados;
-Manipulação básica (criar, consultar, etc);
-Estrutura organizada pensando em API.

🚀 Como rodar
Clone o repositório:
git clone https://github.com/Artrite/reservas
Abra o projeto na sua IDE (ex: IntelliJ ou VS Code)
Execute a aplicação

(dependendo da linguagem/framework, pode precisar rodar via terminal ou buildar o projeto)
